Freally 0.7.20 — Engine enhancement

Resumable scans, diff-rescan, multi-root parallel scanning, battery and wake throttling, and PE/ELF/Mach-O/.NET IL/Java/DEX parsers with a UPX unpacker.

New features

  • Resumable scans — state persists across reboots; resume exactly where you left off.
  • USN file-state diff cache: second scans complete in seconds, not minutes.
  • Multi-root parallel producer/consumer engine with per-mount real-time toggle.
  • Foreground-aware throttle, battery-aware throttle, and AC-only/idle-only wake gate.
  • Format-aware parsers: PE/Authenticode, ELF hardening inventory, Mach-O code-signature, .NET IL extractor, Java bytecode, Android DEX/AXML, UPX unpacker.
  • Sparse-aware hashing, APFS clone dedup, Btrfs/ZFS reflink handling, reparse-point policy, FastCDC selective rehash.
  • Entropy heatmap, packer identification, archive-bomb guard, hot-zone mmap hashing.
  • Remote-mount scan mode (SMB/NFS) with slow-mode throttle.
  • 638 engine unit tests green.

Freally 0.7.13 — Malware & goodware databases

The first public data release — a 53.7 M-row malware blacklist and a 72 M-row NSRL goodware allowlist as SQLite artifacts, with Bloom + Cuckoo filter front-ends and an in-app IOC editor.

New features

  • 53.7 M-row abuse.ch malware hash blacklist (MalwareBazaar + ThreatFox) as a sorted .bin artifact.
  • 72 M-row NIST NSRL RDSv3 goodware allowlist, per-OS sliced, with nightly refresh workflow.
  • Bloom filter + Cuckoo filter front-ends collapse per-file lookup to sub-microsecond for clean files.
  • BLAKE3 + SHA-256 dual-key gate on P0 promotions; hash de-aging and feed-freshness widget.
  • In-app user-IOC editor supporting CSV, STIX 2.1, and MISP formats.
  • Confidence-graded findings; per-finding citation copy; reverse-lookup and lateral-search.
  • Mirror failover with chained Ed25519 epoch signatures; package-manager and App Store allowlists.
