Mythodikal Anti-Virus 0.7.20 — First app release

First app release — per-OS installers + free ed25519 signed auto-updater. The desktop app ships for Windows, macOS, and Linux with behavioral heuristics, archive scanning across 12 formats, a scan scheduler, supply-chain and process-integrity checks, and PE/ELF/Mach-O/ .NET/Java/DEX parsers with a UPX unpacker — an engine that now passes over 800 tests.

What's included

  • Signed installers for every desktop OS — Windows (.exe / .msi), a macOS universal .dmg, and Linux .deb / .rpm / AppImage for both x86_64 and ARM64.
  • A free, ed25519-signed auto-updater so the app keeps itself current.
  • Resumable scans, diff-rescan, multi-root parallel scanning, and battery / wake throttling.
  • Behavioral heuristics, archive scanning across 12 formats, a scan scheduler, and supply-chain and process-integrity checks.
  • Zero telemetry, no accounts, no cloud — every scan stays on your machine.
